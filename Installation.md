## Debian

```bash
$ sudo $apt_pref update
$ sudo $apt_pref install git-extras
```

## Mac

```bash
$ brew install git-extras
```

## Clone / Tarball:

```bash
$ make install
```

One-liner:

```bash
$ (cd /tmp && git clone --depth 1 https://github.com/tj/git-extras.git && cd git-extras && sudo make install)
```
